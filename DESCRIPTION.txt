ОПИСАНИЕ СДЕЛАННОЙ РАБОТЫ

1) Общая идея
Мы сделали простую программу на Python + pandas, которая читает датасет про студентов
(оценки и посещаемость) и автоматически формирует отчёт по успеваемости и посещаемости.

2) Принцип работы программы
- Сначала мы загружаем CSV-файлы из папки data.
- Дальше приводим данные к удобному виду.
- Потом считаем:
  • средний балл каждого студента,
  • топ-10 и анти-топ-10 по среднему баллу,
  • процент посещаемости каждого студента,
  • самые и наименее посещаемые занятия.
- В конце собираем всё в один текстовый отчёт и сохраняем его в report.txt.
  Также сохраняем сводную таблицу students_summary.csv.

3) Распределение работы в команде
ИМЯ1 — загрузка и подготовка данных:
- файл load_data.py (чтение CSV, объединение данных по Student_ID),
- файл clean_utils.py (простые функции для очистки).

ИМЯ2 — успеваемость:
- файл analysis_scores.py (средний балл, топ/анти-топ студентов).

ИМЯ3 — посещаемость:
- файл analysis_attendance.py (посещаемость студентов, самые/наименее посещаемые занятия).

ИМЯ4АВЫЛЫВФЛ — итоговый отчёт и запуск:
- файл report.py (сбор текста отчёта),
- файл main.py (вызывает все функции и сохраняет файлы).
