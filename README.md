## Что делает программа
Программа читает данные о студентах из CSV-файлов и формирует отчёт:
- средний балл каждого студента
- 10 самых успешных и 10 наименее успешных студентов
- посещаемость каждого студента
- самые и наименее посещаемые занятия

Отчёт выводится в консоль и сохраняется в файл `report.txt`.

Также сохраняется таблица `students_summary.csv` (сводка по каждому студенту).

## Как запустить
1. Установите pandas (если не установлен):
   ```bash
   pip install pandas
   ```
2. Запуск:
   ```bash
   python main.py
   ```

# Индивидуальная правка: список кому нужно внимание

## Что добавлено
Добавлен файл attention_list.py

Он делает простой мини отчет по студентам и делит их на 3 группы
1) Низкий балл и низкая посещаемость
2) Низкий балл, но посещаемость нормальная
3) Балл нормальный, но посещаемость низкая

## Зачем это нужно пользователю
В основном отчете есть топы отдельно по баллу и отдельно по посещаемости,
но когда преподаватель ищет кому помочь, ему важно видеть сочетание двух вещей сразу,
поэтому этот мини отчет быстрее отвечает на вопрос кому уделить внимание

## Как запускать
1) поставить pandas
   ```bash
   pip install pandas
   ```

2) запуск
   ```bash
   python attention_list.py
   ```

Программа спросит пороги по баллу и по посещаемости,
если нажать enter, она возьмет средние значения по всем студентам

В конце можно сохранить отчет в attention_report.txt
