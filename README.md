## Что делает программа
Программа читает данные о студентах из CSV-файлов и формирует отчёт:
- средний балл каждого студента
- 10 самых успешных и 10 наименее успешных студентов
- посещаемость каждого студента
- самые и наименее посещаемые занятия

Отчёт выводится в консоль и сохраняется в файл `report.txt`.

Также сохраняется таблица `students_summary.csv` (сводка по каждому студенту).

## Как запустить
1. Установите pandas (если не установлен):
   ```bash
   pip install pandas
   ```
2. Запуск:
   ```bash
   python main.py
   ```
